# restful-web-service

* This is Project for Design Simple Restful Service and deploy with MSA.

## Dependencies

* This project build with Maven
* JDK version : JDK 11
* SpringBootVersion : 2.5.2
* Packaging : Jar

  ### Dependency : 
    * Spring Boot DevTools
    * Lombok
    * Spring Web
    * Spring Data JPA
    * H2 Database
